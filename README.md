# README

## Ruby version

- Ruby `3.3.5`
- Rails `8.0.2`

## System dependencies

- Bun `>= 1.2.x`
- SQLite3 `>= 3.43.x`

## Setup

```bash
bin/setup
```

## Running the app

```bash
bin/dev
```

## Running tests

```bash
bin/rails test
```

Or including integration tests:

```bash
bin/rails test:all
```
